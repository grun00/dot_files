nvim
====

* Leader key is " " (space)

========
```vimscript
 <leader>ga :Git add .<CR>
 <leader>gb :Gblame<CR>
 <leader>gc :Gcommit -v -q<CR>
 <leader>gd :Gdiff<CR>
 <leader>gl :Gllog<CR>
 <leader>gps :Gpush
 <leader>gpl :Gpull
 <leader>gs :Gstatus<CR>
 <leader>gw :Gwrite<CR>
 <leader>gm :Merginal<CR>
 <leader>gr :G<CR>
 <leader>gi :Gread<CR>
 <leader>gt :GV<CR>
```

* Split Vertical Prefix /

ZSH
===

* ctrl + e to open command in vim mode for editing
