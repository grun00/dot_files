nvim
====

* Leader key is " " (space)

Fugitive
========
```vimscript
<leader>ga :Git add .<CR>
<leader>gb :Gblame<CR>
<leader>gc :Git commit -v -q<CR>
<leader>gd :Gdiff<CR>
<leader>go :Gllog<CR>
<leader>gs :Gstatus<CR>
<leader>gm :Git merge
<leader>gps :Gpush origin HEAD<CR>
<leader>gpl :Gpull
<leader>gw :Gwrite<CR>
<leader>gcb :Merginal<CR>
<leader>gr :G<CR>
<leader>gt :GV<CR>
```
